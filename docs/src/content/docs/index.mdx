## Features

<CardGrid stagger>
  <Card title="gRPC Protocol" icon="setting">
    Implements the `omnia.runtime.v1.RuntimeService` protocol for seamless integration with Omnia.
  </Card>
  <Card title="LangChain Integration" icon="puzzle">
    Uses LangChain and LangGraph for powerful agent orchestration capabilities.
  </Card>
  <Card title="Multiple Providers" icon="rocket">
    Supports Claude, OpenAI, Gemini, and Ollama out of the box.
  </Card>
  <Card title="Session Management" icon="document">
    Memory and Redis session stores for conversation persistence.
  </Card>
  <Card title="Tool Support" icon="add-document">
    HTTP and MCP tool adapters for extending agent capabilities.
  </Card>
  <Card title="PromptPack Support" icon="open-book">
    Uses PromptPack for flexible prompt configuration.
  </Card>
</CardGrid>
